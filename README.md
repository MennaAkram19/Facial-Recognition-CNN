# Facial Recognition with CNNs

This project implements a Convolutional Neural Network (CNN) to classify faces using the Labeled Faces in the Wild (LFW) dataset.

## Dataset
Labeled Faces in the Wild (LFW)

## Model
The CNN architecture includes:

- Convolutional layers
- Batch Normalization
- ReLU activation
- Max Pooling
- Global Average Pooling
- Dropout

## Training
The model was trained for 20 epochs using the Adam optimizer.
